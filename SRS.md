### 2. ملخص المشروع

نظام تجارة إلكترونية متكامل باللغة العربية (فصحى) موجه للسوق الجزائري، يدعم الدفع عند الاستلام (COD) بدون تسجيل إجباري. يتميز بواجهة نظيفة بألوان Navy/Coral مع لوحة تحكم كاملة لإدارة المخزون والطلبات.

**السعر المقترح: 280,000 دينار جزائري (~ 2,000 دولار)**

---

### 3. المكدس التقني

| الطبقة | التقنية |
|--------|---------|

### 6. هيكل قاعدة البيانات (النماذج الرئيسية)

```
Stock (المخزون)
├── product_name, description, price, quantity
├── category (اختيارات محددة مسبقاً)
├── sizes, colors (JSON نصي)
├── product_image (صورة رئيسية)
├── sales_count, created_at, updated_at
└── relation → ProductImage (معرض صور)

Orders (الطلبات)
├── ref_code (كود مرجعي)
├── customer_name, customer_phone
├── wilaya, commune, address
├── selected_size, selected_color, quantity
├── status (حالة الطلب)
├── total_price
└── relation → Stock

ProductImage (صور المنتج)
└── image, stock (ForeignKey)
```

### 7. تدفق العمل الأساسي

```
1. الزائر يصل إلى الصفحة الرئيسية ← يتصفح المنتجات
2. يفتح صفحة تفاصيل المنتج ← يختار مقاس/لون/كمية
3. يملأ نموذج الطلب (الاسم، الهاتف، الولاية، البلدية، العنوان)
4. يضغط "أرسل الطلب" ← يظهر SweetAlert تأكيد
5. التاجر يستلم إشعار في لوحة التحكم (SSE)
6. التاجر يغير حالة الطلب: قيد المعالجة → تم التأكيد → في التوصيل → مكتمل
7. عند الإلغاء ← استعادة الكمية للمخزون تلقائياً
```
